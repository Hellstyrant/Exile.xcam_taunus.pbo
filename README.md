# Exile.xcam_taunus.pbo

What's included:
A stock Exile mission file, customized for the Xcam Taunus map. 

What's not included:
Server files, custom scripts, loot tables, loot positions, etc.

How to install:
Download, pack into a .pbo, upload to your MPMissions folder.
